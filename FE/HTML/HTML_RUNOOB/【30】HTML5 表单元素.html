HTML5 新的表单元素 HTML5 有以下新的表单元素: input>标签定义选项列表。请与 input 元素配合使用该元素，来定义 input 可能的值。 keygen> 标签规定用于表单的密钥对生成器字段。 output> 标签定义不同类型的输出，比如脚本的输出。 注意:不是所有的浏览器都支持HTML5 新的表单元素，但是你可以在使用它们，即使浏览器不支持表单属性，仍然可以显示为常规的表单元素。
